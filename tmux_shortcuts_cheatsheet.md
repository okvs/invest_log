# tmux 단축키 정리

## 1. 세션(Session) 관련
| 기능 | 단축키 / 명령 |
|---|---|
| 새 세션 만들기 | `tmux new -s 세션이름` |
| 세션 분리(detach) | `Ctrl + b` → `d` |
| 세션 목록 확인 | `tmux ls` |
| 세션 접속(attach) | `tmux attach -t 세션이름` |
| 세션 종료 | `tmux kill-session -t 세션이름` |
| 모든 세션 종료 | `tmux kill-server` |

## 2. 창(Window) 관련
| 기능 | 단축키 |
|---|---|
| 새 창 생성 | `Ctrl + b` → `c` |
| 다음 창 이동 | `Ctrl + b` → `n` |
| 이전 창 이동 | `Ctrl + b` → `p` |
| 특정 창 번호 이동 | `Ctrl + b` → `숫자` |
| 창 목록 보기 & 선택 | `Ctrl + b` → `w` |
| 창 이름 변경 | `Ctrl + b` → `,` |

## 3. 패인(Pane) 관련
| 기능 | 단축키 |
|---|---|
| 세로 분할 | `Ctrl + b` → `%` |
| 가로 분할 | `Ctrl + b` → `"` |
| 패인 전환(화살표 키) | `Ctrl + b` → 방향키 |
| 다음 패인 | `Ctrl + b` → `o` |
| 패인 닫기 | `Ctrl + b` → `x` |
| 패인 크기 조절 | `Ctrl + b` → `Ctrl + 방향키` |

## 4. 기타 유용 단축키
| 기능 | 단축키 |
|---|---|
| 마지막 창/패인으로 이동 | `Ctrl + b` → `l` |
| 명령 모드 실행 | `Ctrl + b` → `:` |
| 복사 모드 진입 | `Ctrl + b` → `[` |
| 붙여넣기 | `Ctrl + b` → `]` |
