매수 종목 매수 기록

종목명, 섹터, 수량, 매수가, 매수 근거를 한 번에 입력합니다. 종목코드는 네이버 금융에서 자동 검색됩니다.

# 1. "매수" 입력 후 안내 메시지 수신 삼성전자 반도체 10주 72000원 AI 수요 증가 전망 --> 종목 검색 중... --> 매수 기록 완료! 삼성전자 [005930.KS] (반도체) 10주 x 72,000원 = 720,000원
부분 검색
"삼성" 입력 시 삼성전자, 삼성전기, 삼성SDI 중 선택
닉네임
"삼전" 입력 시 삼성전자로 자동 변환
추가 매수
같은 종목 재매수 시 평균단가 자동 계산
종목코드 캐시
한번 검색된 종목은 즉시 조회

매도 매도 기록 + 회고

보유 종목을 선택하고 매도 정보를 입력합니다. 수익률이 자동 계산되며, 매도 후 투자 회고를 진행할 수 있습니다.

# 1. "매도" 입력 → 보유 종목 버튼 표시 --> [삼성전자 | 10주] [카카오 | 5주] # 2. 종목 선택 후 매도 정보 입력 5주 85000원 목표가 도달 --> 매도 기록 완료! 삼성전자 5주 x 85,000원 = 425,000원 수익: +65,000원 (+18.1%) --> 매도 회고를 시작할까요? [회고 시작] [나중에]

회고 5단계:

  1. 투자 판단이 맞았는가? (맞았다 / 틀렸다 / 부분적으로)
  2. 잘한 점은?
  3. 아쉬운 점은?
  4. 피할 수 있었는가? (피할 수 있었다 / 통제 불가 / 모르겠다)
  5. 이번 매매에서 얻은 교훈은?

수정 보유 종목 수정

잘못 입력한 매수 정보를 수정합니다. 종목명, 섹터, 수량, 평균단가, 매수 근거를 변경할 수 있습니다.

# 1. "수정" 입력 → 보유 종목 카드 표시 --> [삼성전자 | 반도체 | 10주 | 평균 72,000원] [카카오 | IT | 5주 | 평균 55,000원] # 2. 종목 선택 → 현재 정보 + 복사용 텍스트 표시 # 3. 복사 → 수정 → 붙여넣기 삼성전자 반도체 10주 70000원 AI 반도체 수요 증가 --> 수정 완료!
"종목명: 삼성전자" 형식으로 붙여넣기해도 라벨을 자동 제거하여 인식합니다.

닉네임 종목 닉네임 관리

# 등록 닉네임 삼전 : 삼성전자 --> 닉네임 등록: 삼전 → 삼성전자 # 삭제 닉네임 삼전 : 삭제 # 목록 조회 닉네임 --> 삼전 → 삼성전자 카뱅 → 카카오뱅크
